# RF to USB Dongle

**Installation of Kicad on Ubuntu**

```bash
sudo add-apt-repository --yes ppa:kicad/kicad-5.1-releases

sudo apt update

sudo apt install --install-recommends kicad
```

**After installation;**

```bash
cd ~/Desktop

git clone "https://github.com/aliozcanunal/RF-Design"

cd /RF-Design/Rf -Design

kicad Rf -Design.pro
```
